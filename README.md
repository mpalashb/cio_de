## cio_de Python scraper

### About

cio_de is the scrapy project. You get the data for top 500. Available fields for this spider

> (Rank, Name, Branche)

### Requirements

Nothing else just

> pip install scrapy

### Running

```
git clone https://github.com/mpalashb/cio_de
cd cio_de
scrapy crawl cio -o out.csv
```
